# o-pi

Orion's Pi Agent.

## 安装使用

克隆到 Pi 配置路径:

```bash
git clone https://github.com/Orion-zhen/o-pi.git ~/.pi
```

安装依赖:

```bash
cd ~/.pi && npm install
```
